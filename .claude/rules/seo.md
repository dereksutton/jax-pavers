# SEO Rules
- Every page must export `metadata` with unique title, description, and openGraph properties
- Canonical domain is jaxoutdoorspaces.com (no www)
- Include structured data (JSON-LD) appropriate to page type
- H1 must be unique per page — only one H1 per page
- Maintain heading hierarchy: H1 > H2 > H3 (no skipping levels)
- Alt text required on all images — descriptive, keyword-relevant
- Internal linking between related service pages
- NAP (name, address, phone) must be consistent across all pages
